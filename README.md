# AutoDriveRL

Autonomous driving using Reinforcement Learning and DonkeyCar.

## Get Started

1. Install Pipenv
2. Run `cd src/`
3. Install the simulator and third party dependencies with `./setup.sh`
4. Test the simulator by running `pipenv run python test.py`
